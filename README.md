# Tareas App

App fullstack para gestión de tareas con autenticación.

##  Demo
https://tareas-lmowmkbey-lautaro-mendez-portelas-projects.vercel.app/
##  Tecnologías
- React
- Node.js
- Express
- MongoDB
- Tailwind

##  Features
- Login / Register
- Crear tareas
- Editar tareas
- Eliminar tareas
- Marcar como completadas
